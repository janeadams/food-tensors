\section{Software Contribution}

All code for this project is intended to live in the public repository \texttt{food-tensors}~\cite{food_tensors_repo}, available at \url{https://github.com/janeadams/food-tensors}.
The repository contains the full software pipeline used by this paper: scripts to ingest and validate canonical Cube Rule examples, generate candidate foods for empty tensor cells with an LLM, filter and review those candidates, render static and interactive tensor figures, and compile the manuscript PDF.
It also contains a small local web application that serves the generated interactive figures through a browser interface, so the tensor projections, 3D occupancy views, and supporting figures can be inspected outside the paper PDF.
In that sense, the repository is not merely an appendix to the paper; it is the executable form of the ontology, the data-processing pipeline, and the figure-viewing workflow described in the surrounding sections.
