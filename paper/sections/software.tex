\section{Software Contribution}

This project includes a full software pipeline for the paper: scripts to ingest and validate canonical Cube Rule examples, generate candidate foods for empty tensor cells with an LLM, filter and review those candidates, render static and interactive tensor figures, and compile the manuscript PDF.
It also includes a small local web application that serves the generated interactive figures through a browser interface, so the tensor projections, 3D occupancy views, and supporting figures can be inspected outside the paper PDF.
In that sense, the codebase is not merely an appendix to the paper; it is the executable form of the ontology, the data-processing pipeline, and the figure-viewing workflow described in the surrounding sections.
